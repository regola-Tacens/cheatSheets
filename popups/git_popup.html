COMMANDES GIT

Create a Repository

Creates a new local repository

git init [project_name]

Download from an existing repository

git clone my_url

Observe your Repository

List new or modified files not yet commited

git status

Show the changes to files not yet staged

git diff

Show the changes to staged files

git diff --cached

Show the changes between two commit ids

git diff commit1 commit2

List the changes

git blame [file]

show the file changes for a commit id and/or file

git show [commit]:[file]

show full change history

git log

show full change history

git log -p [file/directory]

Working with Branches

List all branches

git branch

List all branches local & remote

git branch -av

Switch to a branch, my branch, and update working directory

git checkout my_branch

Create a new branch called new_branch

git branch new_branch

Merge branch_a into branch_b

git checkout branch_b

git merge branch_a

Tag the current commit

git tag my_tag

Make a Change

Stages the file, ready for commit

git add [file]

Stages all changed files, ready for commit

git add .

Commit all staged files to versioned history

git commit -m"message"

Unstages files, keeping the file changes

git reset [file]

Revert everything to the last commit

git reset --hard

Temporary Commits

Save modified ans staged changes

git stash

List satck-order of stashed file changes

git stash list

Write working from top of stash stack

git stash pop

Discard the changes from top of stash stack

git stash drop

Synchronize

Add a git URL as an alias

git remote add [alias] [url]

Get the latest changes from origin

git fetch

Fetch the latest changes from origin and merge

git pull

Fetch the latest changes from origin and rebase

git pull --rebase

Push local changes to the origin

git push
