Quantifiers Anchors Characters Groups & range Patterns

REGEX

Anchors

^ Start of string, or start of line in multi-line pattern

^dog

begins with "dog"
$ End of string, or end of line in multi-line pattern

dog$

ends with "dog"
^  Start of string      $  End of string

^dog$

is exactly "dog"
\<  Start of word      \>  End of word

\<.*\>

will match any string bounded by word characters

Character Classes

\s  White space

\s*

any number of whitespace characters
\S   No White space

/\S/g

global search for non-whitespace characters in a string
\d   Digit

/\$\d

matches a string that has a $ before one digit
\D   No digit

/\D/g

Do a global search for digits ( same result as [^0-9] )
\w   matches a word character (alphanumeric character plus underscore)

\w

Outputs the same result as [A-Za-z0-9_]
\W   not word

\w

Outputs the same result as [^A-Za-z0-9_]

Quantifiers

*   0 or more

abc*

matches a string that has ab followed by zero or more c
+   1 or more

abc+

matches a string that has ab followed by one or more c
?   0 or 1

abc?

matches a string that has ab followed by ero or one c
{3}   Exactly 3

abc{2}

matches a string that has ab followed by 2 c
{3,}   3 or more

abc{2,}

matches a string that has ab followed by 2 or more c
{3,5}   3, 4 or 5

abc{2,5}

matches a string that has ab followed by 2 up to 5 c

Groups and Ranges

.   Any character except new line (\n)

<.+?>

matches any character one or more times included inside < and >
(a|b)   a or b

^.*\b(one|two|three)\b.*$

matches a complete line of text that contains any of the words “one”, “two” or “three”
(...)   group a part of the regular expression together. This allows you to apply a quantifier to the entire group

Set(Value)?

matches Set or SetValue
(?...)   Non capturing group

a(?:bc)*

using ?: we disable the capturing group
[abc]   Range (a or b or c)

[a-zA-Z]

a through z or A through Z, inclusive (range)
[^abc]  Not (a or b or c)

[^abc]

any character _except_ a, b, or c (negation)
[a-q]  Lower case letter from a to q
[A-Q]  Upper case letter from A to Q
[0-7]  Digit from 0 to 7

Pattern modifiers

g   Global match

/D/g

matches all non-word characters globally
i   Case-i­nse­nsitive

/aBc/i

would match AbC
m   Multiple lines

/^\d/gm

takes a digit from the beginning of each line
s   Treat string as single line
